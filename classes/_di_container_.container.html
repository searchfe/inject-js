Class Container

依赖注入容器

根据控制反转的概念： Inversion of Control (IoC)，Container 作为控制反转的容器，当你给容器提供一个 token 时，容器会自动的根据这个 token 值去注入对应的依赖，而这需要 @inject 和 @injectable 去生成 metadata。
